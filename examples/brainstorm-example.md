# 示例：头脑风暴

## 用户输入
我们想提升新用户首周留存，帮我 brainstorm。

## 推荐处理

### 1. 问题重述
新用户在注册后 7 天内没有形成核心行为或价值感知，导致流失。

### 2. 发散方向
- Onboarding：首日引导、任务清单、样例数据。
- 价值感知：首次成功体验、结果反馈、成就提示。
- 激励机制：新手任务、积分、权益券。
- 触达策略：站内信、邮件、短信、push。
- 个性化：按来源/角色推荐不同路径。
- 人工辅助：高价值用户客服/运营介入。

### 3. 收敛
- 立即可做：新手任务 + 首次成功反馈。
- 值得验证：按用户角色个性化引导。
- 长期探索：自动化生命周期运营。
- 暂不建议：复杂成长体系，成本高且验证慢。

### 4. 验证指标
- D1/D7 留存
- 首次核心行为完成率
- Onboarding 完成率
- 触达点击率
- 退订/投诉率作为护栏
